# CampusShop GT

## :pencil: Descripción
CampusShop es una plataforma de comercio electrónico diseñada para la venta de ropa,
enfocada en una experiencia moderna, elegante y accesible. Este proyecto busca simular
un entorno real de e-commerce, integrando diseño responsivo, navegación intuitiva y
funcionalidades clave de compra.

## :rocket: Vista previa del Portafolio
:point_right: https://canuxantonio502.github.io/CampuShop/index.html

## :hammer_and_wrench: Tecnologías utilizadas
- HTML5
- CSS
- Git & GitHub

## :sparkles: Características
- Catálogo de productos
- Sistema de carrito de compras
- Página de perfil de usuario
- Interfaz de Checkout
- Página de contacto
- Diseño responsivo
- Estilo minimalista y elegante
